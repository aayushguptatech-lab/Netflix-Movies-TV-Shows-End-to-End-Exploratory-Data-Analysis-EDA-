# Netflix-Movies-TV-Shows-End-to-End-Exploratory-Data-Analysis-EDA-

This project is a complete exploratory data analysis of Netflix's global content library, covering 8,807 titles released between 1925 and 2021. The analysis goes beyond surface-level charts — it investigates the business logic behind what Netflix has built, where the catalog is structurally misaligned with how subscribers actually behave, and what the data suggests the company should do differently.
The notebook walks through every stage of a real analytical workflow: data quality assessment, outlier reasoning (statistical vs. logical), missing value treatment, and a full suite of analysis including univariate, bivariate, multivariate, time series, director frequency, and cast cross-role analysis. Each section closes with an observation grounded in the numbers, not just a chart.
Key findings include the dominance of TV-MA ratings and its implications for emerging market expansion, the 1,793 single-season TV shows that represent a hidden subscriber churn risk, India's position as Netflix's most represented talent pool by raw title count, the 90-minute runtime concentration in the movie catalog, and the July content addition peak that points to an untapped programming calendar strategy.
Tools used: Python, Pandas, NumPy, Matplotlib, Seaborn
Dataset: Netflix Movies and TV Shows — Kaggle
